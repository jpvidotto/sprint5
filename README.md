Este projeto apresenta um aplicativo web interativo desenvolvido com Streamlit, voltado para a exploração e visualização de dados de anúncios de veículos. A aplicação permite analisar o conjunto de dados de forma dinâmica, utilizando gráficos interativos que auxiliam na interpretação dos dados e na identificação de comportamentos e relações relevantes.

Os dados analisados incluem informações como o valor dos veículos, a quilometragem registrada e outros atributos presentes nos anúncios.

Tecnologias Utilizadas

Python

Streamlit

Pandas

Plotly Express
